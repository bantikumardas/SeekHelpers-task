# 📝 To-Do List Backend API

A simple Node.js + Express API for managing a to-do list, using an in-memory datastore.

This project is part of the MERN/Nest/Node.js Intern Assessment.

---

## 🚀 Features

✅ Fetch all tasks  
✅ Add a new task  
✅ Update (toggle) task completion  
✅ Delete a task  
✅ Basic validation

---

## ⚙️ Tech Stack

- Node.js
- Express.js
- CORS
- Body-parser
